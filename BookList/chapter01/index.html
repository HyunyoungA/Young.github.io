1장. 컴퓨터 내부의 언어 체계

컴퓨터는 어떤말을 사용할까



1.언어란 무엇인가


비트는 2진법을 사용한다. 비트상자에는 모스 부호의 점과 선처럼 두 가지 기호 중 하나만 담을 수 있다.
자연어와 마찬가지로 기호의 순서가 중요하다.
기호라는 개념은 추상적이다. 실제로는 기호가 무엇이든 관계없지만 언어는 문맥 없이는 제대로 작동할 수 없다.
비트는 척하기를 상당히 많이 사용한다. 비트들이 파란색을 뜻한다고 가정하자! 라는 식을 자주 볼 것이다.
프로그래밍이 실제로 이런 식으로 작동한다. 따라서 표준적인 비트 사용법을 배우더라도 자신에게 적합한 자신만의
비트 사용법을 만들어내는 것을 두려워할 필요없다.

논리연산


다른 비트들이 표현하는 내용으로부터 새로운 비트를 만들어내는 이런 동작을 논리연산이라고 한다.
기본적인 불리언 연산자는 NOT, AND, OR 세가지에 +XOR이라는 합성 연산이 있다.
NOT는 논리적 반대의 의미다.거짓에 NOT을 하면 참이고, 참에 NOT을 하면 거짓이다.

AND는 둘 이상의 비트에 작용되는데 모두 참이어야 참이고, 하나라도 거짓이면 거짓이다.

OR는 둘 이상의 비트에 작용한다. 2비트 중 한 비트라도 참이면 결과도 참이다.

XOR(배타적 exclusive OR)는 두개의 비트가 첫번째 비트와 두번째 비트가 다른 값을 가진 경우에 참이 된다. 참+참은 거짓이다.

드모르간의 법칙

a and b는 NOT(NOT a OR NOT b)와 같다.
긍정적인 논리에 더해 부정적인 논리를 기술하는 명데를 사용할 때 활용할 수 있다.

정수를 비트로 표현하는 방법

양의 정수 표현


우리는 보통 10진수체계를 사용한다. 10의 거듭제곱으로 표현하여 사용할 수 있다.

10³ 10² 10¹ 10⁰
5 0 2 8

(base-10) 예를들어 5028을 2진수로 표현한다면,

2¹² 2¹¹ 2¹⁰ 2⁹ 2⁸ 2⁷ 2⁶ 2⁵ 2⁴ 2³ 2² 2¹ 2⁰
1 0 0 1 1 1 0 1 0 0 1 0 0

1001110100100이라는 2진수를 표현한다. 4096+512+256+128+32+4=5028이다.
10진수는 4자리 숫자이고, 2진수는 13비트 수이다.

2진수 덧셈

10진 덧셈은 오른쪽(가장 작은 유효 숫자)에서 왼쪽으로 각 자릿수를 서로 더한다.
결과가 9보다 크면 1을 올린다. 2진수는 각 비트를 LSB에서 MSB쪽으로 더하여 결과가 1보다 크면 1을 다음자리 왼쪽으로 올린다.(0=>10)
진수이 덧셈은 두 비트를 서로 더한 결과는 두 비트를 XOR한 값과 같고, 올림은 두 비트를 AND 한 값과 같다.
